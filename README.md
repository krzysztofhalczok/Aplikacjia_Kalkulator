# Aplikacjia_Kalkulator
**Kalkulator Uniwersalny (Terminal Application)**

To jest aplikacja terminalowa z umiejętnościami obliczania geometrii 2D oraz 3D, obliczania podstawowych funkcji oraz ze standardowego kalkulatora. Aplikacja jest dostępna w języku polskim i została stworzona z myślą o szybkich obliczeniach inżynierskich i matematycznych w środowisku konsolowym.

**Główne Funkcje**

**1. Kalkulator Standardowy**

Podstawowe operacje arytmetyczne.

Obsługa liczb zmiennoprzecinkowych (z użyciem kropki jako separatora).

**2. Kalkulator #Geometryczny (2D i 3D)**

Geometria 2D: Obliczanie pól i obwodów figur (m.in. trójkąt, prostokąt, trapez, równoległobok).

Geometria 3D: Obliczanie objętości i pól powierzchni brył (m.in. ostrosłupy, graniastosłupy).

**3. Kalkulator Funkcji**

Analiza funkcji kwadratowych i liniowych.

Wyznaczanie miejsc zerowych.

Obsługa przypadków szczególnych (np. gdy współczynnik $a = 0$).

**Cechy Systemu**

Interfejs: Tekstowy (konsolowy), zintegrowany z systemem podpowiedzi dla użytkownika.

Pamięć Krótka: Możliwość wykorzystania wyniku poprzedniej kalkulacji poprzez wpisanie specjalnej wartości ($2137.69$).

Lokalizacja: Pełna obsługa języka polskiego z dbałością o poprawność terminologiczną.
